# Agent Instructions

This project uses agent skills for UI work, prompt engineering, and writing style.

## Skills

### simple-english

Use simple, easy English in every reply and in all project text. Read and follow:

`.agents/skills/simple-english/SKILL.md`

Use short sentences. Use common words. Do not use dashes in prose. Do not use fancy words.

### frontend-design

When building, styling, or redesigning web UI (HTML, CSS, JavaScript, pages, components, or layouts), read and follow:

`.agents/skills/frontend-design/SKILL.md`

Apply its design thinking, typography, color, motion, and layout guidelines. Avoid generic AI aesthetics.

### prompt-engineering-patterns

When optimizing prompts, improving LLM output, designing prompt templates, or debugging prompt issues, read and follow:

`.agents/skills/prompt-engineering-patterns/SKILL.md`

Apply its patterns for few-shot learning, chain-of-thought, structured outputs, and production prompt design.
